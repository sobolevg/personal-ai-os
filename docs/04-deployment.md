# Deployment

Date: 2026-07-06  
Status: Deployment notes only. No production deployment has happened.

## Target Model

The VPS should eventually contain a checkout of this repository:

```text
/opt/personal-ai-os
```

Hermes should load personal OS capabilities from this repository or from an
installed package generated by this repository.

## Release Flow

```text
Git repository -> reviewed commit -> tagged release -> deployed to VPS
```

## Deployment Guardrails

- No direct permanent edits on the VPS.
- No Hermes restart without a written rollback command.
- No secrets committed.
- Back up Hermes config before changing deployment state.
- Verify Telegram and Notion behavior after deployment.

## Future Deployment Checklist

1. Pull the selected release tag on the VPS.
2. Run validation.
3. Back up current Hermes config and OS state.
4. Install or update OS plugin/tools.
5. Restart Hermes only after validation.
6. Verify Telegram health.
7. Verify Notion write with dry-run or a test page.
